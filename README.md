# Academic 学术主页 - 后台管理系统

Vue 3后台管理系统，用于管理学术主页内容。

## 技术栈
- Vue 3.4.0
- Element Plus 2.5.0
- Vue Router 4.2.5
- Axios 1.6.2
- Vite 5.0.0

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

访问地址: `http://localhost:8081`

### 3. 构建生产版本
```bash
npm run build
```

## 功能模块
- 登录认证
- 首页管理
- 新闻管理
- 成员管理
- 科研成果管理
- 项目管理
- 相册管理
- 招聘管理

详细文档见主目录 README.md
